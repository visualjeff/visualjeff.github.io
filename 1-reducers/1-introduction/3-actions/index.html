Adding Actions

Let’s add more functionality to our reducer. Add two actions to the counterReducer:

  • RESET: Sets count to 0
  • SET: Sets count to a specific value provided in action.payload

Followed by adding two buttons to the jsx to support the new reset and set actions”

  • RESET: Dispatches a RESET action with no payload.
  • SET: Dispatches a SET action with a payload of 10.
